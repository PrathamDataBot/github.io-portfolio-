# Pixel Desktop Portfolio

An interactive portfolio site built as a retro 16-bit pixel-art desktop OS — boot screen, draggable windows, a wandering pixel character, sound effects, and hidden easter eggs, all in place of a normal scrolling page.

**Live demo:** _(add your GitHub Pages URL here once deployed)_

## What's inside
- Animated parallax scene (sky, hills, pipes, question blocks, castle) as the desktop background
- 15 clickable icons — About, Projects, Experience, Skills, Resume, GitHub, LinkedIn, Contact, AI Projects, Data Analytics, Machine Learning, BlendTrend, Photography, Achievements, Certificates — each opening a draggable window with real content
- Retro sound effects (Web Audio API, no audio files), a coin counter, day/night toggle, and a mute toggle
- Easter eggs: a hidden warp pipe, a Konami code, and a secret credits room
- Mobile view swaps the desktop for a tappable "map" of the same sections
- Single self-contained `index.html` — no build step, no external dependencies

## Tech
Vanilla HTML, CSS, and JavaScript. Pixel art is drawn with CSS `box-shadow` grids rather than image assets; sound is synthesized live via the Web Audio API.

## Running locally
Just open `index.html` in a browser. No install, no server required.
